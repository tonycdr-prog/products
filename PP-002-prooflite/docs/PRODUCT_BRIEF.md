# ProofLite — Product Brief

## What it does
Generates lightweight social proof widgets (testimonials + logo wall) with clean HTML/CSS, no iFrames.

## Target users
- SaaS founders and marketers
- Agencies building landing pages

## Core workflow
1. Add testimonials + logos
2. Choose layout
3. Copy‑paste embed code

## Pricing strategy
- Free: 1 widget
- Pro: £3/mo per widget
- Agency: £25/mo unlimited widgets

## Competitive edge
Performance‑first widget (no trackers), portable, simple embed code.
