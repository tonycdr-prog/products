# FlyerForge — Product Brief

## What it does
Simple, web‑based desktop‑publishing replacement for Microsoft Publisher. Create flyers/brochures and export to PNG/PDF.

## Target users
- Schools, charities, community groups
- SMEs needing quick print materials

## Core workflow
1. Choose template
2. Edit text/images/colors
3. Export to PNG/PDF

## Pricing strategy
- Starter £6/mo (5 exports)
- Studio £18/mo (unlimited exports)
- Agency £40/mo (teams + templates)

## Competitive edge
Faster than InDesign, more print‑friendly than Canva, Publisher‑like simplicity.
