# NoteForge — Product Brief

## What it does
NoteForge is a lightweight, affordable Evernote-style note vault with built‑in OCR search. Users can capture notes, attach images, and search across text + OCR without paying heavy subscription fees.

## Target users
- Freelancers and small teams
- Anyone who wants “second brain” storage without Evernote pricing
- Professionals scanning business cards, receipts, or quick snapshots

## Core features (MVP)
- Fast note capture + tags
- OCR extraction from images
- Full‑text search across notes and OCR
- Local storage (offline‑first) + optional cloud sync later

## Pricing strategy
- Free: 50 notes, basic search
- Personal: £3/mo, unlimited notes + OCR
- Team: £8/mo, shared vaults + admin

## Why now
Evernote pricing keeps rising. Many users want a smaller, faster tool that only does the essentials but keeps OCR search.
