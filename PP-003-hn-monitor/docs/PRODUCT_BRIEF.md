# PRODUCT_BRIEF — SignalHunt (PP-003)

## What it does
SignalHunt monitors new Hacker News posts and scores them against example pains you provide. It highlights high-similarity posts, lets you filter by signals (recency, points, comments), and generates reply drafts so you can engage early.

## Target users
- Indie founders and early-stage SaaS teams
- Developer tool makers hunting for high-intent discussions
- Agencies managing multiple products

## Core features (MVP)
- Paste example pain statements
- HN scan (last N hours) via Algolia API
- Similarity scoring with keyword boosting
- Filters: points, comments, include/exclude keywords
- Result table with CSV export
- Reply draft template per post
- Preset saving (local storage)
- Waitlist capture

## Pricing strategy
- Starter: £19/mo (1 channel, hourly scans)
- Pro: £49/mo (15-min scans, Slack/Telegram)
- Agency: £129/mo (multi-client dashboards)

## Differentiation
- Optimized for HN (not generic social listening)
- Simple similarity scoring without paid embeddings
- Actionable outputs (reply drafts + export) in one screen
