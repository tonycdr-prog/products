**Title:** DNS migration? Run a 30‑second email deliverability pre‑flight (no login)

**Body:**
I kept seeing client migrations break email because one record was missing, so I built a tiny pre‑flight audit you can run before a nameserver change.

- Checks MX, SPF, DKIM, DMARC
- Gives a score + fix checklist
- Exports a JSON report for handover
- Runs entirely in the browser, no login

Try it: https://tonycdr-prog.github.io/products/dns-email-audit/
Feedback welcome — especially if there’s a record check you want added.
