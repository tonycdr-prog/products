# PRODUCT_BRIEF — AI TDD Workbench

## What it does
AI TDD Workbench converts plain-English product requirements into:
- Acceptance criteria
- Test cases (Jest/Vitest or Gherkin)
- Implementation task checklist
- Reusable AI prompt for deeper refinement

This version is fully static and runs 100% in-browser with no backend. Users can save/load locally, export Markdown specs, and export JSON session files.

## Target customer
- Solo developers and small engineering teams shipping quickly
- Agencies that need repeatable QA planning across client features
- Product engineers who want structured TDD artifacts before coding

## Pricing model
- **Solo (£19/mo):** unlimited local generations, template packs
- **Team (£79/mo):** shared projects, cloud sync, collaboration
- **Pro CI (£249/mo):** policy checks in PR/CI, audit logs, governance templates

## Competitive advantage
- Faster than blank-page planning: requirement → tests in one flow
- Works offline and requires zero setup
- Practical outputs (Markdown + JSON + checklist), not just generic AI text
- Designed for pre-code alignment, reducing rework and regression risk
