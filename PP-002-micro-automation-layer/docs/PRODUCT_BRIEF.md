# Product Brief — Micro‑Automation Layer

**What it does:**
Turns manual copy‑paste workflows into a simple automation recipe + checklist that non‑technical owners can implement.

**Who it’s for:**
UK micro‑businesses and agencies using 3–10 tools who can’t justify a full automation platform.

**Core value:**
Reduce admin time by 2–5 hours/week without heavy setup.

**Pricing strategy:**
£15/mo starter, £39/mo pro (templates + shared recipes).
