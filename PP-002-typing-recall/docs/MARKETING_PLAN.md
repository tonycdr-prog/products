# TypeTrace — Marketing Plan

## 5 distribution channels
1. Hacker News (Ask HN follow‑up)
2. Reddit: r/productivity, r/writing, r/sysadmin
3. Chrome/desktop app directories (future)
4. SEO: “typed text history”, “recover typed text”, “clipboard alternative”
5. Privacy communities (local‑first forums)

## First-week launch tactics
- Day 1: Ask HN “built a safe typed‑text recall demo — thoughts?”
- Day 2: Reddit demo post with GIF
- Day 3: Landing page SEO + blog post
- Day 4: Collect feedback; add top feature requests to roadmap
- Day 5: Offer early‑access discount to waitlist
