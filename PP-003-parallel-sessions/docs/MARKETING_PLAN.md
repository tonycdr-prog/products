# Marketing Plan — Parallel Sessions Pilot

## Channels
1. HN/Reddit devtools threads (Ask HN, r/LLMdev, r/programming)
2. GitHub README badges in agent CLI repos
3. Twitter/X clips showing session switching + handoff copy
4. IndieHackers devtools audiences
5. Product Hunt mini-launch

## First-week launch tactics
- Post a demo GIF showing focus lane + handoff export
- Comment on Ask HN threads about coding agents with a link to the pilot
- Publish a blog post: “Why 2–3 lanes beats 6 tabs”
- Offer 20 free pro seats for feedback interviews
- Collect 10 user interviews to refine prompt templates

- "Stop losing context when you run multiple agents."
- "Handoff notes + timers keep every session shippable."
- "Copy a ready-to-run prompt in one click."
