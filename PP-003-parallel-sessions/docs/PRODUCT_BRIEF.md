# Product Brief — Parallel Sessions Pilot

## What it does
A lightweight orchestration layer for running multiple coding-agent sessions in parallel without losing context. Users create focus slots, attach goals, store handoff notes, and export summary prompts for each lane.

## Target users
- Solo developers using Claude Code/OpenCode/Pi
- Small dev teams experimenting with coding agents
- Founders who switch contexts across multiple repos

## Core MVP features
- Create/manage session lanes with status, priority, and slots (local storage)
- Focus lane view with handoff notes + next steps + prompt template
- Per-lane focus timer and status updates
- Copyable handoff summary + JSON export
- Waitlist capture + pricing tiers

## Pricing strategy
- Starter £19/mo (individuals)
- Pro £39/mo (power users)
- Team £79/mo (small teams)
