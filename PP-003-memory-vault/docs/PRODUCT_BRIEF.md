# Product Brief — MemoryVault

## What it does
Encrypts/decrypts AI memory files locally in the browser using Web Crypto (AES‑GCM). No uploads.

## Who it’s for
- Individuals using Claude/agent memory files
- Teams sharing prompt/agent memory repositories
- Security‑conscious orgs

## Pricing strategy
- Free: local encrypt/decrypt
- Team (£8/user/mo): shared vault, access policies, audit logs

## Core value
Simple, safe encryption without CLI friction.
