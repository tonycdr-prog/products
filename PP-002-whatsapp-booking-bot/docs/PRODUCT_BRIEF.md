# PRODUCT_BRIEF — WhatsApp Booking Bot

## What it does
Turns WhatsApp conversations into a self‑serve appointment flow. Customers choose services and times via WhatsApp; staff receive confirmed bookings.

## Who it’s for
Small businesses that take bookings via WhatsApp (salons, clinics, trades, tours).

## Core features
- Bot flow builder (services + hours)
- Booking link integration
- Auto‑confirmations + reminders

## Pricing strategy
Starter £25/mo, Pro £49/mo, Team £99/mo

## MVP scope
Front-end bot flow generator + copy templates. Backend WhatsApp integration in v2.
