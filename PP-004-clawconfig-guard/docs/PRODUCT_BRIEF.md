# Product Brief — ClawConfig Guard

## What it does
Schema-aware editor for OpenClaw settings.json with automatic backups, version history, and rollback.

## Target users
- OpenClaw operators
- Agent framework users struggling with config corruption

## Core MVP features
- JSON validation against required keys
- Version history + rollback
- Waitlist capture and pricing

## Pricing strategy
- Solo £12/mo
- Pro £29/mo
- Team £59/mo

## Next build steps
- Pull OpenClaw JSON schema automatically
- Git-based backup + diff view
- CLI integration (watch file + auto-backup)
