# Marketing plan

## Channels
1. Reddit value posts in the source communities, no spam.
2. SEO landing pages for long-tail problem keywords.
3. Short LinkedIn demos aimed at UK SMEs/property professionals.
4. Partnerships with accountants/property managers/leaseholder groups.
5. Free calculators submitted to directories and relevant newsletters.

## First-week tactics
- Ship static page on Netlify/Vercel.
- Post a transparent “I made a free checker after seeing these threads” update.
- DM 10 commenters only if allowed and with a useful free result.
- Add analytics and track calculator completions.
- Set up Stripe payment links once 10+ waitlist signups arrive.
