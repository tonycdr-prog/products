# ScheduleFlow — Product Brief

## What it does
ScheduleFlow is a lightweight scheduling tool that recalculates delivery dates daily based on real progress. Teams can lock a baseline schedule, log today’s progress/blockers, and see drift instantly without heavyweight PM suites.

## Target users
- Small dev agencies (2–20 people)
- Product teams using spreadsheets for delivery tracking
- UK SMEs that need schedule clarity without complex tooling

## Core features (MVP)
- Task input with estimates and completed days
- Daily progress + blockers to adjust remaining workload
- Baseline schedule snapshot + drift calculation
- Risk flag (low/moderate/high)
- CSV import/export + local save in browser

## Pricing strategy
- Starter £19/mo (3 projects)
- Team £49/mo (unlimited projects)
- Agency £99/mo (client workspaces)

## Value prop
“Know your real end date every day, without running a full PM stack.”
