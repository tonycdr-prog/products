# MARKETING_PLAN — TypeBack (PP-002)

## 5 Distribution Channels
1. Hacker News “Show HN” + productivity threads
2. Reddit r/productivity + r/macapps (value-focused demo)
3. YouTube productivity creators (short demo clips)
4. GitHub README integration for dev tools (sidecar snippet recall)
5. SEO content: “recover typed text”, “keyboard history”

## First-week launch tactics
- Day 1: Post demo + waitlist
- Day 2: 2-min walkthrough video
- Day 3: Ask 10 power users for feedback
- Day 4: Offer beta invites for referrals
- Day 5: Launch on Indie Hackers
