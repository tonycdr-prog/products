# LessonLedger — Product Brief

## What it does
LessonLedger is a lightweight tutor operations tool that tracks:
- Lesson credits purchased per client
- Lessons delivered over time
- Payments received and remaining balance
- Ready-to-send statement + invoice nudge text

It runs as a single offline HTML file with local browser storage, so tutors can use it immediately without setup.

## Target customer
Independent UK tutors and small tutoring teams who:
- Sell lessons in packs
- Manually track balances in notebooks/spreadsheets
- Lose revenue due to under-billing or delayed top-ups

## Pricing model
- **Starter (£15/month):** up to 25 active clients, CSV export
- **Pro (£29/month):** unlimited clients, templates, cloud sync
- **Team (£59/month):** multi-tutor workspace, shared client records

## Competitive advantage
- **Offline-first, zero setup:** can open and use in <1 minute
- **Built for lesson-credit businesses:** not generic invoicing software
- **Action-oriented output:** statement + nudge copy generated from live ledger data
