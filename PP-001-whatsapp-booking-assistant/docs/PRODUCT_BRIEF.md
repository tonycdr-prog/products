# PRODUCT_BRIEF — WhatsApp Booking Assistant

## What it does
Lightweight booking workflow for tour and service businesses using WhatsApp. Captures booking details, generates confirmation + voucher templates, and exports a daily CSV.

## Who it’s for
UK tour operators, experience providers, and small service businesses that rely on WhatsApp for bookings.

## Core features
- Booking intake form (date, time, headcount)
- WhatsApp confirmation template generator
- Voucher + sold-out reply templates
- CSV export for daily ops

## Pricing strategy
- Starter £19/mo (100 bookings)
- Pro £39/mo (unlimited + multi-staff)
- Agency £99/mo (multi-location)

## MVP scope
HTML/JS front-end demo + template generator. Backend integration with WhatsApp Web or official API in v2.
