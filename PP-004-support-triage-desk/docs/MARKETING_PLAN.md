# MARKETING_PLAN — TriageDesk

## Landing page headline
**Triage support tickets in 60 seconds — and handle revenue-risk cases first.**

## 3 core benefits
- **Prioritise what matters:** Auto-rank by urgency, account value, and issue type.
- **Respond faster:** SLA targets + ready-to-send response drafts for every ticket.
- **Run support like an operator:** Export queue CSV and copy standup-ready summaries.

## CTA copy
**Start triaging today (free beta)**

## One distribution channel
**Founder-led LinkedIn + IndieHackers demo posts**
- Post weekly “before vs after triage” clips showing raw inbox → prioritised queue
- Include one free template/sample queue and direct users to the beta waitlist
