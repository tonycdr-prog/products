# PRODUCT_BRIEF

## Product
DiagramPolish — a browser tool that auto‑styles architecture diagrams into presentation‑ready visuals.

## Target Users
- Engineering leads, architects, consultants
- Teams writing docs/diagrams for stakeholders

## Problem
Manual polishing takes time; diagrams look inconsistent across teams.

## Solution
Paste Mermaid, apply smart theme + spacing presets, export a clean SVG/PNG.

## MVP Scope
- Mermaid rendering + theme presets (clean/neutral/forest/dark)
- Flow direction + spacing controls
- SVG/PNG export + copy SVG
- Local save of last diagram + settings
- Waitlist capture

## Pricing
- Free: 3 exports/month
- Pro: £29/mo
- Teams: £99/mo
