# SoloSignal — Product Brief

## What it does
SoloSignal turns a solo founder’s marketing goal into a 7‑day execution plan, channel task board, and KPI tracker. It’s a focused planning workspace you can run in a browser without a full CRM or agency overhead.

## Who it’s for
- Solo technical founders (SaaS/devtools/B2B services)
- Small teams without a marketer
- UK founders who need focus + speed

## Core features
- Goal-based weekly plan generator (7‑day cadence)
- Channel task board with editable tasks + status
- KPI tracker with baseline/target/actual status
- Exports: CSV task list + JSON plan
- Local save/load in browser

## Pricing strategy
- Starter £19/mo (single channel planning)
- Builder £39/mo (multi‑channel + exports + KPI tracker)
- Studio £99/mo (multiple products + team seats)

## Differentiation
- Narrow scope (one goal, realistic time budget)
- Tasks sized for solo founder weeks
- KPI guardrails baked into planning flow
- Works offline in a single file
