# PRODUCT_BRIEF — Planning Alerts Desk

## What it does
Planning Alerts Desk helps UK residents, tenants’ groups, and local campaigners track planning applications in one place. Users can add application references, councils, validated dates, comment windows, and notes; the tool calculates comment deadlines, flags overdue/soon alerts, and generates copyable summaries.

## Target customer
- Primary: UK residents monitoring nearby developments
- Secondary: small resident associations and local planning consultants handling multiple applications

## Pricing model
- Free: up to 10 tracked applications, local-only storage, CSV/JSON export
- Pro (£9/month): unlimited applications, hosted sync, reminder emails, saved report templates
- Team (£29/month): shared workspace, role-based access, portfolio reporting

## Competitive advantage
- Immediate utility in-browser (no signup needed)
- Deadline-first workflow tailored to planning comment windows
- Offline-friendly and exportable, so users can start with zero setup and still produce usable outputs
