# TypeBack — Product Brief

## What it does
TypeBack is a privacy-first typed text recall tool. It stores recent typing locally so users can recover lost snippets without relying on clipboard history.

## Target users
- Developers, writers, analysts
- Anyone losing typed text due to crashes or accidental deletes
- Support teams who type repetitive replies

## Core features (MVP)
- Local typed snippet history (split on Enter or idle)
- One-click copy to clipboard
- Clear history + retention controls
- Desktop-first roadmap (system-wide input capture)

## Pricing strategy
- Free: last 50 snippets
- Pro: £5/mo unlimited history + encrypted sync
- Team: £12/mo shared patterns + admin

## Notes
Browser MVP demonstrates interaction only. A production version would use native desktop capture with explicit user consent.
